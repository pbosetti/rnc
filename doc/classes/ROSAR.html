Class ROSAR
In: rosar.rb
Parent: Object

Usage example: 

  r=ROSAR.instance
  a=[1,2,3]
  b=[4,5,6]
  r.transfer :a=>a, :b=>b
  r.plot :x=>:a, :y=>:b, :typ=>"'l'"
  r.grid

Example based on dataframes: 

  df = {
    :x => [1,2,3,4],
    :y => [7,2,5.5,8]
  }
  r.data_frame :df, df
  r.attach :df
  r.plot :x=>:x, :y=>"y/2", :typ=>"'b'", :xlab=>"'Time (s)'"
  r.grid
  r.abline :h=>[2.5,3.5]
  r.detach :df
